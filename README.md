# レベルリンク・カードバトル

HTML/CSS/JavaScriptのみで動くローカル版のWebカードゲームです。

## 起動

`index.html` をブラウザで開くか、ローカルサーバーで配信します。

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

その後、`http://127.0.0.1:4173/index.html` を開きます。

## スマホ単体で遊ぶ

PCのローカルサーバーで開く場合、PCをシャットダウンするとスマホからは使えません。

スマホ単体で遊びたい場合は、このフォルダのファイルをGitHub Pages、Netlify、Cloudflare PagesなどのHTTPSで公開できる静的ホスティングに置きます。

HTTPSで一度開くとPWAとして動作します。スマホのブラウザで「ホーム画面に追加」を行うと、アプリのように起動できます。初回読み込み後は、HTML/CSS/JavaScriptが端末にキャッシュされるため、ローカル対戦とAI戦はオフラインでも動作します。

注意: オンライン対戦はFirebaseなどのサーバー同期が必要なため、オフラインでは使えません。

## 実装済み

- メニュー画面
- 初回チュートリアル、遊び方画面
- デイリーミッション、達成報酬
- 第1弾・第2弾ガチャによるカード入手
- ガチャ玉によるガチャ実行
- ガチャ1パックはガチャ玉1個で5枚、10パックはガチャ玉10個で50枚
- ガチャ玉の入手: AI戦勝利、AI戦敗北、1日1回ボーナス
- AI戦の降参ではガチャ玉を入手できない
- カードランク N/R/SR/SSR/UR
- 既存カードと第2弾カードの縦長カードアート
- カード属性
- 第2弾キャラクターカードの召喚時・攻撃時効果
- 吸収効果
- 高ランクカードの低確率排出
- カード創造チケットの消費枚数に応じたカード作成ランク
- 山札編成、100〜300枚デッキ、複数デッキ保存、localStorage保存
- 所持枚数までしか山札に入れられない制限
- カード作成、自動コスト計算、localStorage保存
- ローカル1台2人対戦
- AIとバトル
- AI戦・クエスト用の自動戦闘ON/OFF
- GoogleログインとFirebaseによる部屋IDオンライン対戦MVP、同期revisionによる古い状態の上書き防止
- ゲーム内プレイヤー名の変更、クラウド保存、オンライン対戦・オークション表示への反映
- 余りカードとゴールドを使うカード強化・進化
- クエスト、初回クリア報酬、ゴールド
- ショップ、入札時ゴールド預かり式オークション、ユーザー出品、返金、売上受け取り、カード売却
- 先攻・後攻ルーレット
- ターン制、毎ターンドロー、使用可能コスト
- キャラクター召喚、召喚酔い、1ターン1回攻撃
- 呪文カードのドロー、最大コスト増加、HP回復
- 墓地送りによるレベルアップ
- レベル倍率による攻撃力・防御力補正
- 直接攻撃、キャラクター同士の相互ダメージ
- ブロック能力
- スマホ向けレスポンシブCSS
- PWA対応、HTTPS公開時のホーム画面追加、オフラインキャッシュ
